# MileTrack Version 1.2 (Build 6) - Release Notes

## What's New

### 🎨 Visual Improvements
- **Enhanced Light Mode** - Glass cards now have proper depth and contrast in light mode for better readability
- **Modern Typography** - Added Manrope variable font for hero numbers and section headers
- **Refined Animations** - Smoother pulse animations for tracking status indicator
- **Updated App Icon** - Fresh new icon design

### ⚡ Performance & Accuracy
- **Faster Trip Detection** - Reduced detection delay by ~30 seconds through optimized speed monitoring
- **More Accurate Start Locations** - Trip starting points now backtrack to pre-detection location to capture early miles that were previously lost
- **Precise Trip Addresses** - Improved geocoding accuracy for both start and end locations
- **Better Merged Trips** - Full street addresses now preserved when combining trips

### 💎 Premium Features
- **Enhanced Reports** - Collapsible trip details with paginated expansion for easier navigation
- **Improved CSV Export** - Better formatting with all trip details included
- **Premium Benefits Screen** - Clear overview of Pro subscription features

### 🎯 User Experience
- **4-Page Onboarding** - Redesigned welcome flow that clearly explains how Auto Mode works
- **Consolidated Settings** - Cleaner settings organization for easier navigation
- **Inbox Empty State** - Helpful call-to-action when no trips are pending
- **Editable Addresses** - Full control to edit any trip's start and end addresses

### 🐛 Bug Fixes
- Fixed iOS 26 compatibility crashes related to glass effects
- Fixed missing end addresses on trips recovered from app termination
- Fixed iPad layout issues
- Fixed tracking health pulse animation glitches
- Improved subscription restore reliability with better error messages
- Enhanced diagnostics logging for troubleshooting
