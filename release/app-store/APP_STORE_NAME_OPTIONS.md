- Device display name stays "{{APP_NAME}}"
- Only the App Store listing name changes

## Trademark Claim Process (Optional)

If you have registered trademark rights to "{{APP_NAME}}":

1. Go to App Store Connect
2. Find the "Name Conflict" or "Trademark Claim" option
3. Submit:

   - Supporting documentation
4. Wait for Apple's review (can take weeks)
