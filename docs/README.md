# Docs

This folder mirrors the operational documentation layout from `Cursor/Template/apple-app-meta-template`.

- `docs/xcode/`: Xcode setup and project management guides
- `docs/ios-data/`: persistence guides (SwiftData / Core Data / CloudKit)
- `docs/guides/`: general engineering workflows

If you want the full upstream template docs, run:

```bash
./scripts/template_sync.sh
```

…then review `_template_upstream/docs/` and copy/adapt what you need.
